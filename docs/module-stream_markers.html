(inner) marker(name, id) → {Promise}

Source:
  • stream_markers.js, line 25
Set a single stream marker. Valid values for the "name" argument are: - global - personal - mentions - channel:id
Example
pnut.marker("channel:18", 15100).then(res => {
 console.log(res);
});
Parameters:
Name Type Description
name string Name of the markable stream
id string | number ID of the post / message
Returns:
Type
Promise

(inner) markers(markerObjects) → {Promise}

Source:
  • stream_markers.js, line 48
Set up to 10 markers at once, like described in the pnut docs.
Example
pnut.markers([
    { name: "channel:18", id: 15100 },
    { name: "channel:19", id: 15101 }
]).then(res => console.log(res));
